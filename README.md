# DonermannSimulator

Im Dönermann Simulator kannst du das Leben eines Dönermannes hautnah erleben. Bediene Kunden und bereite Döner zu. 

Anleitung:
1. Schau dir die Bestellung an
2. Wähle Fleisch aus.
3. Schneide Fleisch runter
4. Klicke Zutaten an
5. Klicke Soßen an
6. Falls vorhanden, füge Getränke hinzu
7. Falls vorhanden, füge Beilagen hinzu
